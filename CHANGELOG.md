## [0.0.1-alpha1] - 2025-01-11

### 新增
- 🚀 **项目初始化**: A.YLM 单图像3D重建与智能导航系统
- 🔧 **SHARP集成**: 集成Apple最新发布的SHARP模型
- 🎯 **核心功能**:
  - 单图像高精度3D重建
  - Vision Transformer特征提取
  - 亚像素级深度估计
  - 绝对尺度3D高斯点云生成
- ⚙️ **体素化增强**:
  - 多重RANSAC地面检测算法
  - 自适应阈值离群点去除
  - 局部区域智能裁剪
  - 1cm高精度网格化
  - Rodrigues公式旋转矩阵计算
- 🏗️ **系统架构**:
  - 端到端处理流程
  - 模型预加载机制
  - 批量处理支持
  - 跨平台兼容性
- 📷 **图像处理**:
  - 95+种图像格式支持
  - RAW格式专业解码
  - Apple设备专用格式(HEIC/HEIF/Live Photos)
  - iPhone镜头畸变校正
- 🔄 **坐标系转换**:
  - OpenCV ↔ 机器人标准坐标系
  - OpenCV ↔ ENU地理坐标系
  - 智能设备导航优化

### 技术规格
- **输入**: 单张RGB图像 (640x480 至 8K)
- **输出**: 3D高斯模型 + 体素化数据
- **性能**: <1秒重建时间，<5秒体素化
- **精度**: 1cm体素网格，绝对物理尺度

## 版本命名约定

- **主版本号**: 不向后兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正
- **预发布版本**: alpha(内部测试)、beta(公开测试)、rc(发布候选)

## 贡献类型

- `Added` 新功能
- `Changed` 对现有功能的更改
- `Deprecated` 即将删除的功能
- `Removed` 删除的功能
- `Fixed` 任何错误修复
- `Security` 安全相关的修复
