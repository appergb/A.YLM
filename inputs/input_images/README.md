# 输入图像目录

将您要处理的图像文件放入此目录中。

## 支持的格式

SHARP支持95+种图像格式，包括：

### Apple设备格式

- HEIC, HEIF, MOV (Live Photos)

### 专业RAW格式

- CR2, CR3, NEF, ARW, DNG, RAF, RW2, PEF, SRW, ORF

### 标准格式

- JPEG, PNG, TIFF, BMP, WebP

### 其他格式

- GIF, PBM, PGM, PPM等

## 注意事项

- 推荐使用正常曝光、无明显模糊的标准照片
- 避免超广角镜头，优先使用标准焦距
- 图像分辨率支持从640x480到8K超高清

## 示例

```bash
# 将图像复制到输入目录
cp your_image.jpg inputs/input_images/
cp your_photo.heic inputs/input_images/
```
