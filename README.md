# A.YLM

Single-image 3D reconstruction and intelligent navigation system based on Apple SHARP model.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)

## Features

- Vision Transformer-based single-image 3D reconstruction
- Intelligent voxelization with 1cm precision for navigation
- Support for 95+ image formats, including professional RAW formats
- GPU acceleration with real-time processing (inference <1 second)
- Ground detection, coordinate transformation, and path planning
- Complete CLI tools and Python API
- Modular design for easy extension

## System Requirements

- Python 3.11+
- PyTorch 2.8.0+
- Open3D 0.18.0+
- CUDA-compatible GPU (recommended)
- 4GB+ RAM

## Installation

```bash
# Clone repository
git clone https://github.com/appergb/A.YLM.git
cd A.YLM

# Install dependencies
pip install -e .
pip install -e ml-sharp/

# Or install development version (includes test dependencies)
pip install -e .[dev]
```

## Quick Start

### Using CLI Tools

```bash
# Run complete workflow: environment check + 3D reconstruction + voxelization
aylm process

# Or run step by step
aylm setup      # Environment setup and check
aylm predict    # 3D reconstruction
aylm voxelize   # Voxelization processing

# View all options
aylm --help
aylm process --help
```

### Using Traditional Scripts

```bash
# Run complete workflow
./run_sharp.sh

# Run step by step
./run_sharp.sh --setup      # Environment check
./run_sharp.sh --predict    # 3D reconstruction
./run_sharp.sh --voxelize   # Voxelization
```

## Output Files

- `*.ply`: 3D Gaussian model generated by SHARP
- `cropped_*.ply`: Local area cropping results
- `voxelized_*.ply`: 1cm voxel grid (ready for path planning)

## Configuration

### Environment Variables

```bash
# Directory configuration
export INPUT_DIR="/path/to/input/images"
export OUTPUT_DIR="/path/to/output/directory"
export SHARP_MODEL_PATH="/path/to/sharp/model.pt"

# Model preloading
export SHARP_MODEL_PRELOADED=1
export SHARP_MODEL_DEVICE="cuda:0"
```

### Custom Parameters

```bash
# Run complete workflow (custom input/output directories)
aylm process --input-dir /path/to/images --output-dir /path/to/output

# Run voxelization only (custom parameters)
aylm voxelize --output-dir /path/to/output
python3 -m aylm.tools.pointcloud_voxelizer input.ply --voxel-size 0.01
```

## Troubleshooting

### Common Issues

1. **Python Version Error**
   ```bash
   python3 --version  # Ensure >= 3.11
   ```

2. **Dependency Installation Failure**
   ```bash
   # CPU version of PyTorch
   pip install torch --index-url https://download.pytorch.org/whl/cpu
   ```

3. **Insufficient Memory**
   - Close other applications
   - Use larger voxel size to reduce precision

4. **CUDA Related Issues**
   ```bash
   # Check CUDA availability
   python3 -c "import torch; print(torch.cuda.is_available())"
   ```

### CI Environment Notes

CI configuration is optimized to handle complex dependencies:

- **PyTorch**: Uses CPU version to avoid CUDA complexity
- **Open3D**: Supports pip and conda installation fallback mechanism
- **Caching**: Uses GitHub Actions cache to speed up dependency installation
- **Layered Testing**: Lightweight code quality checks + full unit tests + optional integration tests

### Troubleshooting

If CI fails:

1. **Dependency Installation Failure**: Check the `scripts/install_deps.sh` script
2. **Test Failure**: Run `pytest` locally to reproduce the issue
3. **Code Quality Failure**: Run `pre-commit run --all-files` to check

### Manually Trigger Integration Tests

Add the `integration-test` label to a Pull Request or use workflow_dispatch to trigger manually.
