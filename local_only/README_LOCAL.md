# Local Only Workspace (Not Synced)

This directory stores files that should stay local and should not be uploaded.

## Suggested categories
- `benchmarks/`: ad-hoc benchmark scripts and outputs
- `agent_notes/`: temporary planning/progress notes
- `research/`: papers, drafts, and local docs
- `test_data/`: local test datasets and samples
- `run_outputs/`: archived run outputs

Keep core project runtime structure at repository root.
